Job Portal Web Application using MERN Stack
Project Overview

This project is a full-stack Job Portal Web Application developed using the MERN stack (MongoDB, Express.js, React.js, Node.js). The platform allows job seekers to search and apply for jobs, while employers can post job openings and manage applications efficiently. The application provides a smooth and secure experience through modern web technologies.

Key Features
1. User Authentication

Secure login and registration system for job seekers and employers.

Authentication implemented using JWT (JSON Web Tokens).

Passwords are encrypted using Bcrypt hashing.

2. Job Listings

Users can browse available job opportunities.

Job data is stored and retrieved from MongoDB database.

Employers can create and manage job postings.

3. Application Management

Job seekers can apply for jobs and track their applications.

Employers can view, manage, and review applications submitted by candidates.

4. Responsive Design

The user interface is designed to work smoothly on desktop, tablet, and mobile devices.

Technology Stack
Frontend

React.js – Building dynamic user interfaces

React Router – Navigation between pages

Bootstrap – Responsive UI design

Backend

Node.js – Server-side runtime environment

Express.js – Backend framework for APIs

MongoDB – Database for storing user and job data

Authentication

JWT (JSON Web Tokens) – Secure user authentication

Bcrypt – Password encryption and security

Media Storage

Cloudinary – Used for uploading and storing images such as profile pictures or company logos.

Deployment

Vercel – Frontend deployment

Render – Backend hosting

MongoDB Atlas – Cloud database service

Installation and Setup
Prerequisites

Before running the project, ensure you have:

Node.js (v22.2.0 or later)

MongoDB Atlas account (or local MongoDB server)

Cloudinary account
